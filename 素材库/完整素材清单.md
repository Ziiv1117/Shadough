# Shadough 素材库完整素材清单

创建日期：2026-07-06  
最近刷新：2026-07-06  
项目范围：`ClockTower_TopdownPrototype` Topdown v0.6 Demo Loop  
当前玩法闭环：灯笼 -> 树影过沟 -> 压板门 -> 钥匙影开锁 -> 玩家影诱敌 -> 最终钟核

## 使用说明

本清单用于整理《剪影师 / Shadough》需要制作、收集、替换或后续扩展的素材。当前阶段以可玩 Demo 的正式表现替换为主，不主动扩展无关功能。

## 风格锁定

当前素材方向统一为“中式像素剪影风”。

执行原则：

- `ClockTower_TopdownPrototype` 后续应表现为中式旧城钟楼 / 鼓楼空间，不再按西式钟楼、通用地下城或暗童话场景处理。
- 像素可读性优先于装饰密度。主路径、阻挡、水渠、门、压板、锁孔、可剪影和最终钟核必须一眼可辨。
- 影子素材应结合皮影、剪纸、墨色块面和纸影边缘，但不能改变影子被剪下时保存的形状、角度、尺寸和碰撞判定。
- UI 使用宣纸、匾额、印章、朱砂、灯笼、剪纸边框和像素中文字体，不使用现代科幻或通用卡通按钮。
- 音频使用纸张、竹片、木门、铜铃、锣鼓、灯芯、幕布、低频钟鸣等中式民间器物质感。
- 所有素材替换都必须服从当前玩法闭环，不主动引入新机制。

### 优先级

| 优先级 | 含义 |
| --- | --- |
| P0 | 当前 Topdown Demo 闭环必须有，优先替换灰盒 |
| P1 | 当前 Demo 的可读性、表现和体验提升 |
| P2 | 后续扩展版本再制作，当前可暂不做 |

### 状态

| 状态 | 含义 |
| --- | --- |
| 已有灰盒 | 场景中已有临时对象或简单 Sprite，需要正式素材替换 |
| 需要正式素材 | 当前功能需要，但没有正式美术、音频或动画 |
| 后续扩展 | 策划中提到或适合未来版本，不属于当前必须完成 |
| 需确认 | 需要在 Unity 里进一步确认对象引用或最终美术规格 |
