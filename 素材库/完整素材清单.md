# Shadough 完整素材清单

最近刷新：2026-07-08
项目范围：`ClockTower_TopdownPrototype` Topdown v0.6 Demo Loop
风格方向：中式像素剪影风
当前玩法闭环：灯笼 -> 树影过沟 -> 压板门 -> 钥匙影开锁 -> 玩家影诱敌 -> 最终钟核

## 文档结构

本文件是素材库根目录的总清单，用来回答三个问题：

1. 整个游戏需要哪些素材。
2. 每类素材应该放在哪个素材库文件夹中。
3. 每个分类文件夹内的详细计划清单在哪里查看。

每个分类文件夹内都有自己的 `素材清单.md`，用于说明该文件夹计划存放哪些素材、每个素材的用途、建议 Unity 导入位置。

## 分类边界

- `01_角色与单位` 只存玩家影子的概念设定参考，用来决定它作为主角一部分时的轮廓、气质和视觉方向。
- `03_影子系统` 存玩家影子的运行机制素材，用来支持 `Q` 剪下自身影子、携带、贴出、诱敌、显示图标和碰撞规则。

## 优先制作顺序

| 顺序 | 内容 | 目标 |
| --- | --- | --- |
| 1 | 主角和灯笼素材 | 让玩家主体替换灰盒并完成基础动作表现 |
| 2 | 地面、水渠、墙体、古树、木梁、钥匙纹样 | 让主场景具备中式旧城钟楼视觉基底 |
| 3 | 树影、梁影、钥匙影、玩家影 | 让核心剪影和贴影机制可读 |
| 4 | 压板、门、锁、最终钟核 | 让机关反馈完整 |
| 5 | Reveal View、按键提示、影子栏、胜利提示 | 让玩家理解操作和当前状态 |
| 6 | 剪影、贴影、开门、开锁、诱敌、钟核启动特效 | 强化关键操作反馈 |
| 7 | BGM、环境声、影子音效、机关音效、敌人音效 | 补齐完整体验反馈 |
