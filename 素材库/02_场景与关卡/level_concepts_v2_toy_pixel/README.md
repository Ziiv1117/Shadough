# 五关低分辨率玩具像素概念图 v2

这组图用于修正 v1 过于精细的问题。

处理规则：

- 每张最终图为 `960x540`。
- 先压缩到 `240x135`，再用最近邻放大。
- 限制为 24 色，制造粗颗粒、玩具感、低分辨率设定图效果。
- 每张图只保留本关核心机制，避免名字和地图内容错位。

关卡对应：

1. `level_01_tree_shadow_river.png`：府邸外树影过河，不包含寻影兽、锁门、压板。
2. `level_02_key_shadow_gate.png`：进入府邸，钥匙影开锁。
3. `level_03_pressure_tower_entrance.png`：进入灯塔/钟塔，压板机关。
4. `level_04_shadow_seeker_lure.png`：寻影兽诱饵空间。
5. `level_05_final_clock_core.png`：最终钟盘房。

这些仍是概念图，不是最终 Unity Tilemap。
