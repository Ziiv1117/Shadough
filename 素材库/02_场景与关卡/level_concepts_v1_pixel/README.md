# 五关像素风概念图 v1

这组图用于确定《剪影师》关卡美术和空间方向，属于参考概念图，不是最终 Unity 可运行 Tilemap。

生成方向：

- 更像素化，更接近可拆 Tilemap 的块面。
- 每张只聚焦一个核心机制。
- 降低官邸细节密度，减少装饰和复杂阴影。
- 保留中式像素剪影风、府邸/灯塔/钟盘主题。

文件对应：

1. `level_01_tree_shadow_river.png`：府邸外树影过河。
2. `level_02_key_shadow_gate.png`：府邸内钥匙影开锁。
3. `level_03_pressure_tower_entrance.png`：进入灯塔/钟塔前的压板机关。
4. `level_04_shadow_seeker_lure.png`：寻影兽诱饵空间。
5. `level_05_final_clock_core.png`：最终钟盘房。

第四关里的寻影兽只作为地图概念占位，不是正式敌人素材。
